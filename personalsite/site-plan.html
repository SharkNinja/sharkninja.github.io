Overview

Purpose

The purpose of this website may seem strange at first, however it can be versatile. For those who are fammiliar with the Discord Application, specifically those are Administrators/Moderators of large servers can use this website as a template or base for there server. This website can be used for a large variety of things, such as displaying key information about the server, the staff, appeals, and applications.

Audience

The audience would be the server members themselves, since they will be the ones using it to find all the information that was listed above.

Content

Home page

This home page will act as a hub for the two child pages. The purpose of the website as a whole is to display key information about server roles, channels, staff, and even rules. This website will try to answer anything regarding those things, and members will visit this website for the information they desire because the information on the website will be unique within its own context.

Images for the Home page

[Roles and Channels]

In this child page, people will be able to find out more about the roles and channels. There will be many confusions amongst members about which roles are what, how to get said roles, which channels are which, and how to access certain channels. As a result, this child page will contain detailed information about each and every role to answer every possible question, concern, or eliminated any type of confusion. Visitors will only come here because the information is unique within it's own context, as every server is unique and different.
